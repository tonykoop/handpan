# Handpan Warm-Up: D Kurd Field Walk

Begin each session with this ding-and-field exploration. No rhythm required.

## Ding breathing exercise
Strike the ding lightly with your left thumb. Listen to the full decay (5–8 seconds).
Breathe in as you strike; breathe out as it fades. Repeat 4 times.

## Field walk (ascending)
```
Ding (D3) → A3 → Bb3 → C4 → D4 → E4 → F4 → G4 → A4
```
Strike each with a light fingertip tone stroke. Pause 2 seconds between notes.
Listen for the resonance of each field before moving to the next.

## Interval pairs (ding + each field)
```
D3+A3  D3+Bb3  D3+C4  D3+D4  D3+E4  D3+F4  D3+G4  D3+A4
```
Each pair creates a different harmonic interval. Listen for consonance vs. tension.

## Pentatonic core (D F G A C)
```
D3 — F4 — G4 — A4 — C4 — D4 (loop)
```
These 5 notes form the pentatonic subset of D Kurd — the most harmonious nucleus.
