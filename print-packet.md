# Handpan Print Packet

## Cover

Project: D Kurd Handpan Root-Mode Prototype  
Status: Private research build packet  
Source workbook: `handpan-design-table.xlsx`  
Primary warning: final handpan tuning is empirical and tuner-led.

## Build Intent

This packet gives the shop enough structure to procure shells, build fixtures, lay out the scale, document forming, tune iteratively, validate measured behavior, and review whether the prototype is ready for broader sharing.

## Root Specifications

| Parameter | Value |
|---|---:|
| Scale | D Kurd |
| Notes | D3, A3, Bb3, C4, D4, E4, F4, G4, A4 |
| Shell diameter | 21 in |
| Shell height | 9.5 in |
| Steel thickness | 1.0 mm |
| Gu port | 3.5 in nominal |
| Material path | nitrided DC04 preferred |

## Shop Sequence

1. Inspect shell pair and document material/geometry.
2. Cut/deburr/finish Gu if not supplied.
3. Build tuning support ring and layout templates.
4. Rough-form Ding and tone fields.
5. Tune top shell in measured passes.
6. Rest and drift-check before shell closure.
7. Bond shells only after coupon and dry-fit tests.
8. Final tune through Gu, rest, retune, and validate.

## Validation Summary

Use `validation.csv` as the control sheet. Required evidence includes cents error for all fundamentals, octave and compound-fifth partial behavior, sustain, cross-talk, Gu response, and post-rest drift.

## Release Gate

Keep private until the measured build log and safety/process choices have been reviewed by a human handpan tuner and metal-forming lead.
