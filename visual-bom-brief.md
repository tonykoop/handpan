# Visual BOM Brief

## Objective

Produce a visual BOM plate for the handpan prototype that shows the shell pair, Gu port, tuning frame, templates, dimple tooling, hammer set, measurement kit, adhesive, protective finish, and case. The plate should communicate what a builder must gather before beginning, while labeling quote-sensitive parts clearly.

## Layout

- Header: `D Kurd Handpan Root-Mode Prototype - Visual BOM`.
- Top row: assembled silhouette or shell pair reference image.
- Main grid: part number, part name, quantity, required spec, source status, estimated cost, and risk note.
- Side strip: measurement/tuning tools, because these are not optional for this project.
- Footer: `Prices and suppliers unverified; confirm current availability before purchase. Tone-field dimensions are layout estimates, not final tuning prescriptions.`

## Image Guidance

Use real supplier photos or shop photos where available. If concept renders are generated, label them as visual placeholders until replaced by actual parts. Avoid imagery that suggests final tone-field geometry is fixed by CAD.

## Critical Callouts

- Pre-drawn shells are recommended for the first build.
- Practice shell or scrap coupon testing is mandatory.
- Tuning measurement kit is part of the build, not a nice-to-have.
- Adhesive and treatment choices require coupon tests and supplier review.
